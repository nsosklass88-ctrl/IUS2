ИУС 2.0 — чистая установка с нуля
=================================

Файлы сайта для GitHub Pages:
1) index.html — кабинет ученика
2) expert.html — кабинет учителя + редактор заданий
3) styles.css — оформление
4) apps-script-config.js — ссылка на Apps Script

Файлы для Google Apps Script:
1) Code.gs — код серверной части
2) appsscript.json — разрешения Google Drive и Google Таблицы

--------------------------------------------------
ЧАСТЬ 1. GOOGLE ТАБЛИЦА И APPS SCRIPT
--------------------------------------------------

1. Создайте новую Google Таблицу.

2. В таблице нажмите:
   Расширения → Apps Script

3. В файле Code.gs удалите старый код и вставьте код из файла Code.gs.
   Нажмите Сохранить.

4. Включите показ appsscript.json:
   Шестерёнка слева → Show "appsscript.json" manifest file in editor.

5. Откройте appsscript.json и замените весь текст содержимым файла appsscript.json.
   Важно: внутри обязательно должна быть строка:
   "https://www.googleapis.com/auth/drive"

6. Нажмите Сохранить.

7. Вверху выберите функцию setupIUS и нажмите Запустить.
   Разрешите доступ.

8. Вверху выберите функцию authorizeIUS и нажмите Запустить.
   Разрешите доступ к Google Drive.

Если будет "Приложение не проверено":
   Дополнительно → Перейти к проекту → Разрешить.

После успешного запуска authorizeIUS в логах должно быть:
   DriveApp разрешён. Тестовый файл создан и удалён: ...

Если снова Access denied: DriveApp:
   - откройте https://myaccount.google.com/permissions
   - удалите доступ у старого Apps Script проекта
   - вернитесь в Apps Script
   - снова запустите authorizeIUS
   - обязательно разрешите пункт про Google Drive.

--------------------------------------------------
ЧАСТЬ 2. РАЗВЕРТЫВАНИЕ APPS SCRIPT
--------------------------------------------------

1. Нажмите:
   Развернуть → Новое развертывание

2. Тип: Веб-приложение.

3. Настройки:
   Выполнять от имени: Меня
   У кого есть доступ: Все

4. Нажмите Развернуть.

5. Скопируйте ссылку веб-приложения. Она должна заканчиваться на /exec.

--------------------------------------------------
ЧАСТЬ 3. НАСТРОЙКА САЙТА НА GITHUB
--------------------------------------------------

1. Откройте файл apps-script-config.js.

2. Вставьте ссылку Apps Script:

window.APPS_SCRIPT_WEB_APP_URL = "https://script.google.com/macros/s/ВАША_ССЫЛКА/exec";

3. На GitHub загрузите/замените файлы:
   - index.html
   - expert.html
   - styles.css
   - apps-script-config.js

4. Откройте сайт и нажмите Ctrl + F5.

--------------------------------------------------
ПРОВЕРКА
--------------------------------------------------

1. На странице учителя войдите или зарегистрируйтесь.
   Код учителя: SAHA2026

2. Нажмите "Проверить подключение".
   Должно быть: works: да, tasks: да, Drive: да.

3. На странице ученика зарегистрируйтесь, запишите короткий ответ и нажмите:
   Отправить работу учителю.

4. На странице учителя нажмите:
   Показать отправленные работы.

5. Откройте работу. Должно появиться аудио.

--------------------------------------------------
ВАЖНО
--------------------------------------------------

После любого изменения Code.gs всегда делайте:
   Развернуть → Управление развертываниями → Изменить → Новая версия → Развернуть

Если аудио не играет:
   1) проверьте, есть ли в строке audioFileId;
   2) запустите authorizeIUS;
   3) сделайте новую версию развертывания;
   4) отправьте новую работу.

Старые строки с ошибками audio_error сами не исправляются.
