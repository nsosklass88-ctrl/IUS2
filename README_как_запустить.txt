ИУС 2.0 — версия с редактированием заданий

Что нового:
1) Учитель может менять задания прямо на странице expert.html.
2) Задания хранятся в Google Таблице на листе tasks.
3) Ученик при входе автоматически получает актуальные задания из Google Таблицы.
4) Сохранение работ, аудио и проверка остаются как раньше: Google Таблица + Google Drive + Apps Script.

КАК ОБНОВИТЬ

1. GitHub Pages
Замените на GitHub эти файлы:
- index.html
- expert.html
- styles.css
- apps-script-config.js

2. Google Apps Script
Откройте Google Таблицу → Расширения → Apps Script.
Полностью замените Code.gs на новый файл Code.gs.
Проверьте appsscript.json: в нём должен быть scope:
"https://www.googleapis.com/auth/drive"

3. Запуск функций
В Apps Script запустите по очереди:
- setupIUS
- authorizeIUS

setupIUS создаст лист works и лист tasks.
authorizeIUS проверит доступ к Google Drive.

4. Новая версия развертывания
После замены Code.gs обязательно:
Развернуть → Управление развертываниями → Изменить → Новая версия → Развернуть.

Настройки веб-приложения:
- Выполнять от имени: Меня
- У кого есть доступ: Все

5. Как менять задания
Откройте сайт → expert.html → войдите как учитель.
Найдите блок «Редактор заданий для ученика».
Нажмите «Загрузить задания».
Измените:
- название;
- короткое название вкладки;
- время подготовки;
- время ответа;
- инструкцию;
- текст / тему / вопросы.
Нажмите «Сохранить задание».

После этого ученик должен обновить страницу Ctrl + F5 или снова войти в кабинет. Новые задания появятся автоматически.

ВАЖНО
Если задания не меняются у ученика:
1) проверьте, что Code.gs заменён;
2) проверьте, что setupIUS запускался после обновления;
3) сделайте Новая версия → Развернуть;
4) на сайте нажмите Ctrl + F5.
